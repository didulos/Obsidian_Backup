## 정의

토큰(token)은 텍스트를 처리 가능한 최소 단위로 분할한 것으로, 단어, 문장 부호, 또는 부분 단어(subword)일 수 있다. 언어 모델은 토큰 단위로 텍스트를 읽고 생성하며, 각 토큰은 고유한 ID로 변환되어 처리된다.

## 왜 중요한가

토큰화(tokenization)는 텍스트를 신경망이 이해할 수 있는 형태로 변환하는 첫 단계이며, 모델의 처리 속도와 정확성에 직결된다. API 비용 계산이나 컨텍스트 윈도우(context window) 제한을 이해할 때도 토큰 수가 핵심이다.

## 비유로 이해하기

책의 문장을 낱말 카드로 자르는 것처럼, 텍스트를 작은 조각으로 나누어 컴퓨터가 읽을 수 있게 준비하는 과정이다.

## 실제 사용 예시

GPT-3.5에서 "Hello, world!"는 3개의 토큰으로 분할되며, ChatGPT API 요금은 입출력 토큰 수에 따라 계산된다. 또한 4K, 16K, 128K와 같은 모델 스펙은 처리 가능한 최대 토큰 수를 의미한다.
