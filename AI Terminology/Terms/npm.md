## 정의

npm(Node Package Manager)은 자바스크립트 패키지 관리 도구로, 재사용 가능한 코드 라이브러리를 중앙 저장소에서 다운로드하고 프로젝트에 설치·관리하는 역할을 한다. Node.js의 기본 패키지 매니저로 제공되며, package.json 파일을 통해 프로젝트의 의존성(dependency)을 명시하고 버전을 관리한다.

## 왜 중요한가

현대 자바스크립트 개발 생태계에서 필수적인 도구로, 수백만 개의 오픈소스 패키지를 활용하여 개발 생산성을 크게 향상시킨다. 의존성 버전 관리를 통해 프로젝트의 재현성과 안정성을 보장한다.

## 비유로 이해하기

npm은 도서관 시스템과 같다. 도서관에서 필요한 책을 찾아 빌려가듯이, npm에서 필요한 라이브러리를 검색해서 프로젝트에 설치하고, 도서관처럼 버전과 반납 기한을 관리한다.

## 실제 사용 예시

`npm install express`로 웹 프레임워크를 설치하거나, `npm run build`로 프로젝트를 빌드할 수 있다. package.json에 명시된 모든 의존성을 한 번에 설치하려면 `npm install`을 실행한다.
