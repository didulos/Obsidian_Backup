## 정의

물리 서버를 가상화하여 독립적인 가상 머신(virtual machine) 환경을 제공하는 클라우드 컴퓨팅 서비스이다. 사용자는 자신의 운영체제, 애플리케이션, 데이터를 완전히 제어할 수 있으며, 필요에 따라 리소스를 유연하게 조정할 수 있다.

## 왜 중요한가

머신러닝 모델 학습, 데이터 처리 파이프라인 운영, 프로덕션 환경 배포 시 안정적이고 확장 가능한 인프라를 저렴하게 구축할 수 있기 때문에 필수적이다.

## 비유로 이해하기

VPS는 한 건물(물리 서버)에 여러 명이 독립적인 방을 임차하는 것과 같다. 각 세입자(가상 머신)는 자신의 공간을 완전히 소유하고 관리하지만, 건물의 자원(전기, 수도)은 공유한다.

## 실제 사용 예시

데이터 엔지니어링 팀이 대규모 데이터 전처리 작업을 실행하거나, 데이터 사이언티스트가 학습 모델 서버를 배포할 때 VPS를 임차하여 독립적인 Linux 환경과 GPU 리소스를 할당받아 사용한다.
