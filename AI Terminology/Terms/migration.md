## 정의

머신러닝 모델, 데이터, 또는 시스템을 한 환경에서 다른 환경으로 이동하거나 업그레이드하는 프로세스이다. 모델 버전 관리, 데이터 이전, 인프라 전환 등 다양한 수준에서 발생한다.

## 왜 중요한가

프로덕션 환경에서 안정적으로 모델을 배포하고, 데이터 손실 없이 시스템을 확장하기 위해 필수적이다. 잘못된 마이그레이션은 서비스 중단이나 데이터 불일치를 초래할 수 있다.

## 비유로 이해하기

회사가 사무실을 이전할 때 직원, 문서, 장비를 새 건물로 옮기는 것처럼, 마이그레이션은 모델과 데이터를 기존 시스템에서 새로운 시스템으로 안전하게 이동시키는 과정이다.

## 실제 사용 예시

기존 온프레미스(on-premise) 데이터베이스에서 클라우드로 마이그레이션할 때, 스키마 변환과 데이터 동기화를 수행한다. 또한 TensorFlow 2.0으로 업그레이드하면서 레거시 모델을 새 프레임워크로 마이그레이션하는 작업도 포함된다.
