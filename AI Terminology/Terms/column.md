## 정의

데이터베이스나 데이터프레임에서 세로 방향으로 배열된 데이터의 집합으로, 같은 속성(attribute)을 가진 값들의 모음이다. 행(row)과 함께 표 형식 데이터의 기본 구조를 이룬다. 각 컬럼은 고유한 이름과 데이터 타입을 가진다.

## 왜 중요한가

머신러닝 모델에서 컬럼은 피처(feature)로 직접 변환되므로, 컬럼의 선택과 전처리가 모델 성능을 크게 좌우한다. 데이터 분석과 처리 작업의 기본 단위이기 때문에 올바른 이해가 필수적이다.

## 비유로 이해하기

엑셀 스프레드시트에서 세로로 쭉 이어진 한 줄의 셀들을 생각하면 된다. 예를 들어 학생 정보 표에서 '이름' 컬럼, '나이' 컬럼, '성적' 컬럼처럼 각각의 속성을 나타낸다.

## 실제 사용 예시

```python
df[['age', 'salary', 'experience']]  # 특정 컬럼들 선택
df['age'].mean()  # 나이 컬럼의 평균값 계산
```
