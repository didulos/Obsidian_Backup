## 정의

인터넷을 통해 온디맨드 방식으로 컴퓨팅 리소스(서버, 저장소, 데이터베이스, 소프트웨어)를 제공하는 IT 서비스 모델이다. 물리적 서버를 소유하지 않고 필요한 만큼만 사용하고 비용을 지불한다.

## 왜 중요한가

ML/AI 모델 개발과 배포에서 확장성(scalability)과 비용 효율성을 제공하므로, 대규모 데이터 처리와 GPU 리소스를 유연하게 활용할 수 있다.

## 비유로 이해하기

자동차를 소유하는 대신 필요할 때마다 렌트카를 빌려서 사용하는 것처럼, 서버를 구매하는 대신 필요한 만큼 빌려서 쓴다.

## 실제 사용 예시

AWS, Google Cloud Platform(GCP), Microsoft Azure 같은 클라우드 서비스에서 머신러닝 모델을 훈련하고 배포할 때 탄력적으로 GPU 클러스터(cluster)를 확장한다. 모델 추론 서버를 클라우드에 배포하여 트래픽에 따라 자동으로 인스턴스(instance)를 증감시킨다.
