## 정의

컴퓨터 프로세서에서 동시에 실행되는 경량의 실행 단위(lightweight execution unit)로, 하나의 프로세스 내에서 메모리와 자원을 공유하면서 독립적으로 작동한다. AI/ML 분야에서는 데이터 로딩, 모델 학습, 추론 등의 작업을 병렬로 처리하기 위해 활용된다.

## 왜 중요한가

GPU/CPU의 멀티코어를 효율적으로 활용하여 처리 속도를 높이고, 특히 대규모 데이터셋 학습이나 배치 처리에서 I/O 대기 시간을 줄여 전체 파이프라인의 성능을 개선한다.

## 비유로 이해하기

식당의 주방을 생각해보자. 한 명의 셰프(프로세스)가 여러 명의 조리사(스레드)에게 지시하면, 각 조리사는 같은 냄비와 재료를 공유하면서도 동시에 다른 음식을 만든다.

## 실제 사용 예시

PyTorch의 `DataLoader`에서 `num_workers` 파라미터로 데이터 로딩 스레드 수를 지정하거나, Python의 `threading` 모듈을 사용해 모델 추론 요청을 병렬 처리할 때 활용된다.
