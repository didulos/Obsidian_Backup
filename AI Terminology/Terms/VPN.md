## 정의

VPN(Virtual Private Network)은 공개 네트워크 위에 암호화된 터널을 만들어 사용자의 트래픽을 보호하는 기술이다. 사용자의 실제 IP 주소를 숨기고 데이터를 암호화하여 안전한 통신을 제공한다.

## 왜 중요한가

ML/데이터 파이프라인 운영 시 민감한 데이터 전송, 온프레미스와 클라우드 간의 보안 연결, 그리고 지역 제한 우회를 통한 안정적인 모델 학습 환경 구축에 필수적이다.

## 비유로 이해하기

VPN은 투명한 우편함에서 편지를 보낼 때, 편지를 상자 안에 넣고 자물쇠를 채운 후 보내는 것과 같다. 누군가가 상자를 가로채도 내용을 볼 수 없고, 발신자 주소도 다른 주소로 표시된다.

## 실제 사용 예시

회사의 프라이빗 데이터 센터에서 AWS 환경으로 학습 데이터를 전송할 때 VPN 터널을 통해 암호화된 연결을 구성한다. 또는 지역 제한이 있는 API에 접근할 때 VPN을 거쳐 요청을 전송한다.
