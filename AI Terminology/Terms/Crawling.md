## 정의

웹 페이지나 API를 자동으로 방문하여 데이터를 수집하는 프로세스이다. 크롤러(crawler)가 링크를 따라다니며 구조화되거나 비구조화된 데이터를 추출한다. 머신러닝 모델 학습을 위한 대규모 데이터셋 구축에 필수적인 기술이다.

## 왜 중요한가

웹에 존재하는 방대한 데이터를 자동으로 수집함으로써 모델 학습에 필요한 데이터셋을 효율적으로 구축할 수 있다. 특히 준지도학습(semi-supervised learning)이나 전이학습(transfer learning) 환경에서 unlabeled 데이터 확보에 핵심적인 역할을 한다.

## 비유로 이해하기

도서관의 사서가 체계적으로 책장을 돌아다니며 필요한 책 정보를 기록하는 것처럼, 크롤러가 인터넷의 웹 페이지를 자동으로 방문하여 정보를 수집한다.

## 실제 사용 예시

뉴스 기사 감정분석(sentiment analysis) 모델 학습을 위해 수백만 개의 뉴스 댓글을 크롤링하거나, 이미지 분류 모델을 위해 검색 엔진으로부터 수십만 개의 레이블된 이미지를 자동으로 수집한다.
