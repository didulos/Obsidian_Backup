## 정의

프레임워크(framework)는 머신러닝 모델을 구축하고 학습하기 위한 소프트웨어 라이브러리와 도구의 통합 집합이다. 신경망 구성, 데이터 처리, 모델 학습, 배포 등의 반복적인 작업을 추상화하여 개발자가 알고리즘 로직에 집중할 수 있도록 한다.

## 왜 중요한가

프레임워크를 사용하면 복잡한 수학적 연산을 직접 구현할 필요 없이 검증된 고성능 코드를 활용할 수 있으며, 생산성과 코드 안정성을 크게 향상시킨다. 또한 대규모 프로젝트에서 팀 협업과 코드 재사용성을 높인다.

## 비유로 이해하기

프레임워크는 요리 키트(cooking kit)와 같다. 재료와 도구가 미리 준비되어 있어서 요리사는 창의적인 레시피 개발에만 집중할 수 있고, 처음부터 냄비와 칼을 만들 필요가 없다.

## 실제 사용 예시

PyTorch를 사용하여 합성곱 신경망(CNN)을 구축할 때, `torch.nn.Conv2d`라는 프레임워크 내 모듈로 복잡한 합성곱 연산을 간단히 구현할 수 있다. TensorFlow의 Keras API를 통해 `Sequential` 모델로 층을 쌓아 올리듯이 신경망을 정의할 수 있다.
