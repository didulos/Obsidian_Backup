# Node.js 개념 정리

## Ask & Output

**Ask:**
node.js란 무엇이지?

**Output:**
Node.js는 Chrome V8 JavaScript 엔진 위에서 동작하는 서버사이드 JavaScript 런타임이다. 이벤트 루프 기반 비동기 I/O로 동시 연결을 효율적으로 처리하며, 브라우저에서만 실행되던 JavaScript를 서버/로컬 환경에서도 사용할 수 있게 해준 기술이다.

## Lessons Learned

- Node.js는 브라우저 전용이던 JavaScript를 서버에서도 실행 가능하게 만든 런타임이며, 핵심 차별점은 이벤트 루프 기반 비동기 I/O 모델이다

## Better Ask

**Before:**
node.js란 무엇이지?

**After:**
Node.js가 무엇인지 알고 싶다. 브라우저 JS와의 차이점, 주요 사용 사례(웹 서버, CLI 도구 등), 그리고 비동기 I/O 모델이 주는 장점을 요약해줘.
